Lista de pruebas para página appProductos:

Ingresar al link: Dominio/appProductos

1. Página HOME muestra la descripción de mi persona, y las plantillas siguientes de Clientes, Productos, Facturas que pertenecen a 
   un header heredado en todos los templates.
2. Pestaña de Clientes, muestra una tabla con los clientes ya registrados en nuestra base de datos, con la posibilidad de borrar, o
   editar tales datos de los mismos. También es posible añadir nuevos clientes, sin la necesidad de entrar al link de administrador.
   A todo esto, se le añadio un formulario de busqueda, donde usted puede buscar cualquier cliente que desee, tan solo con el CUIL.
3. Pestaña de Productos, muestra una tabla con los productos ya registrados en nuestra base de datos, con la posibilidad de borrar,
   o editar tales datos de los mismos. También es posible añadir nuevos productos, sin la necesidad de entrar al link de administrador.
   A todo esto, se le añadio un formulario de busqueda, donde usted puede buscar cualquier producto que desee, tan solo con el nombre.
4. Pestaña de Facturas, muestra una tabla con las facturas ya registradas en nuestra base de datos, con la posibilidad de borrar,
   o editar tales datos de las mismas. También es posible crear nuevas facturas, sin la necesidad de entrar al link de administrador.
   A todo esto, se le añadio un formulario de busqueda, donde usted puede buscar cualquier factura que desee, tan solo con el Número de 
   factura.


Funcionalidades:

Se utilizaron 3 modelos para esta entrega, las cuales contienen clientes, productos y facturas. También se utilizó la herencia de un HTML 
maestro, en cada una de las pestañas. Se utilizó un forms de Django para insertar datos en todas las clases creadas en los modelos, y también
un formulario para buscar estos mismos datos en nuestra BD.
